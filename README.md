# NOVAIX OS

Premium cinematic operating interface for cognitive business infrastructure demos, audits, automation simulations and operational intelligence.

## Run

```bash
npm install
npm run dev
```

## Private access

Create `.env.local` from `.env.example` before running the app. NOVAIX OS will not authenticate anyone unless `NOVAIX_USER`, `NOVAIX_PASSWORD` and `JWT_SECRET` are configured.
